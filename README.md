BuyNow.com – Full Stack E-Commerce Application
Overview
BuyNow.com is a full-stack e-commerce web application designed to demonstrate real-world online shopping functionality using modern web technologies. The application allows users to browse products, manage a shopping cart, place orders, and perform secure authentication, while providing a scalable and maintainable backend architecture.
This project is built with Java (Spring Boot) on the backend and React on the frontend, following industry best practices for REST APIs, layered architecture, and component-based UI development.
________________________________________
Tech Stack
Backend
•	Java
•	Spring Boot
•	Spring MVC
•	Spring Data JPA
•	Hibernate
•	RESTful APIs
•	MySQL (or any relational database)
Frontend
•	React
•	JavaScript (ES6+)
•	HTML5
•	CSS3
•	Axios (for API calls)
Tools & Others
•	Maven
•	Git & GitHub
•	Postman (API testing)
________________________________________
Features
User Features
•	User registration and login
•	Secure authentication
•	Browse product catalog
•	View product details
•	Add/remove products from cart
•	Place orders
Admin / Backend Features
•	RESTful API design
•	Database integration using JPA/Hibernate
•	Layered architecture (Controller, Service, Repository)
•	Exception handling
•	Clean and maintainable code structure
________________________________________
Project Structure
Backend (Spring Boot)
•	controller – Handles HTTP requests
•	service – Business logic
•	repository – Database access layer
•	entity – JPA entities
•	dto – Data transfer objects
•	exception – Custom exception handling
Frontend (React)
•	components – Reusable UI components
•	pages – Application screens
•	services – API integration
•	routes – Application routing
________________________________________
How to Run the Project
Backend Setup
1.	Clone the repository
2.	Open the backend project in your IDE
3.	Configure database details in application.properties
4.	Run the Spring Boot application
Frontend Setup
1.	Navigate to the frontend folder
2.	Install dependencies:
3.	npm install
4.	Start the React application:
5.	npm start
________________________________________
API Testing
•	APIs can be tested using Postman
•	Base URL:
•	http://localhost:8080
