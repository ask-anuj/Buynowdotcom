Overview

BuyNow.com is a full-stack e-commerce web application designed to demonstrate real-world online shopping functionality using modern web technologies. The application allows users to browse products, manage a shopping cart, place orders, and perform secure authentication, while providing a scalable and maintainable backend architecture.

This project is built with Java (Spring Boot) on the backend and React on the frontend, following industry best practices for REST APIs, layered architecture, and component-based UI development.

How to Run the Project
Backend Setup

Clone the repository

Open the backend project in your IDE

Configure database details in application.properties

Run the Spring Boot application

Frontend Setup

Navigate to the frontend folder

Install dependencies:

npm install


Start the React application:

npm start

API Testing

APIs can be tested using Postman

Base URL:

http://localhost:8080
